# Changelog

## [0.2.0] - 2026-06-28

### Added
- **Миграция на CustomTkinter** — полная замена `tkinter`/`ttk` на `customtkinter` со скруглёнными углами, карточками и современными виджетами.
- **Кастомный `FileListBox`** — собственный виджет списка файлов на базе `CTkScrollableFrame` с выделением синим и поддержкой Ctrl+Click.
- **Всплывающие подсказки файлов** — при наведении на файл в списке отображается полный путь в tooltip.
- **Параллельные потоки** — выбор количества параллельных потоков (1–8) через `CTkOptionMenu` с настройкой `max_workers` в конвейере.
- **`Card.TFrame` / `_card_section()`** — хелпер для создания белых карточек с заголовком (замена `ttk.LabelFrame`).

### Changed
- **Все виджеты перенесены на CTk** — `CTkButton`, `CTkEntry`, `CTkLabel`, `CTkTextbox`, `CTkProgressBar`, `CTkCheckBox`, `CTkOptionMenu`, `CTkToplevel`.
- **Цветовая палитра** — обновлена для современного минималистичного стиля (`BG_MAIN`, `BG_CARD`, `FG_LABEL`, `FG_HEADER` и др.).
- **README.md** — обновлён под CustomTkinter, добавлены `customtkinter` и `max_workers` в зависимости/конфигурацию, обновлена mermaid-диаграмма для параллельных потоков.

## [0.3.0] - 2026-06-28

### Added
- **Поддержка PDF файлов** — чтение PDF-документов через `PyMuPDF` (`fitz`) с извлечением текста постранично.
- **OCR-распознавание сканов** — опциональная поддержка `pytesseract` + `Pillow` для распознавания текста на отсканированных страницах PDF. Требует установленный `Tesseract-OCR` на системе.
- **Уведомление об OCR** — центрированная подсказка под кнопками файлового списка, отображается при наличии PDF-файлов и отсутствии Tesseract.
- **Флажок «Только нарезка и конверсия»** — позволяет пропустить шаг обработки LLM: файл нарезается на чанки и сразу упаковывается в `.md` без обращения к модели.
- **Прогресс-бар выбранного файла** — при клике на файл в списке прогресс-бар отображает статус именно этого файла в реальном времени.
- **Кнопка «Удалить промпт»** — удаление выбранного шаблона промпта из библиотеки с подтверждением. Запрещено удалять последний промпт.

### Changed
- **Левая панель** — фиксированная ширина 400px, карточки расширяются на всю ширину.
- **Кнопки файлового списка** — «Добавить», «Удалить», «Очистить» отцентрированы.
- **Фильтр диалога выбора файлов** — добавлен формат `*.pdf`.
- **Имя прогресс-бара** — изменена подпись с «Прогресс текущего файла» на «Прогресс: <имя_файла>».

### Dependencies
- Добавлены: `PyMuPDF`, `pytesseract`, `Pillow`
